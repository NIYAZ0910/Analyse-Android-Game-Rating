# Android Game Analysis

## Project Overview
This project analyzes Android game data to understand trends in ratings, installs, category performance, and growth. The goal is to identify top-performing game categories and provide insights useful for game developers and marketers.

## Dataset
The dataset contains information about Android games including:
- Ratings
- Installs
- Game category
- Growth trends
- Paid vs free status

## Objectives
- Analyze game performance across multiple metrics
- Identify categories attracting the most users
- Understand trends in ratings and installs
- Compare paid vs free game performance

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization
- Dashboard Creation

## Key Insights
- Some categories dominate installs and ratings
- Paid games perform differently compared to free games
- Growth patterns vary across categories
